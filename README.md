# BASIC CRUD APP USING Mern-stack

### <i>Clone the project</i>

## symlink environment
ln -sF $(pwd)/env/.env.example $(pwd)/.env

## Backend
<p>cd backend</p>
<p>npm install</p>
<i> Run server and connect Db </i>
<p>node server</p>

## Frontend
<p>cd views</p>
<p>npm install</p>
<i> Build and run the app </i>
<p>npm start or yarn dev</p>
